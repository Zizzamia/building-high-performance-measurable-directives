Now I want show you the way we do in ngTasty

We have a Yeoman generator help us to create benchmark

Create benchmark of specific bower version

npm install generator-benchpress

yo benchpress:external ng-tasty#0.5.0

Run benchmark

gulp build.js.cjs

gulp serve.js.prod serve.js.dart2js

protractor protractor-js.conf.js --spec ng-tasty-0-5-0 --benchmark

What about if I’m debugging my directive and see the difference in run-time, we notice that to run a benchmark you need call

  • benchpress build
  • benchpress run
